# Crypto Data Analytics Dashboard

## Overview

A real-time Cryptocurrency Analytics Dashboard built using Python, CoinMarketCap API, Pandas, and Power BI.

The project automatically collects cryptocurrency market data every 5 minutes for 24 hours, stores the data in CSV format, and visualizes key market insights through an interactive Power BI dashboard.

---

## Features

* Real-time cryptocurrency data collection
* Automated ETL pipeline using Python
* Data backup and recovery mechanism
* Market capitalization analysis
* Trading volume monitoring
* Top gainers analysis
* Interactive Power BI dashboard
* Time-series cryptocurrency tracking
